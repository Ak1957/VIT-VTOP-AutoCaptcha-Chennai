VIT VTOP AutoCaptcha Chennai

AutoCaptcha for new VTOP Chennai by Ayush

Features -
-> Generates captcha for the new VTOP site for chennai campus.
-> Works even if captcha is refreshed.
-> Works after login fails also.
-> Can save username and password locally if the user wishes to.
A confirmation alert window will open after first successfull login.
Credentials will be saved if user accepts it.
If declined, it no further alert box will open up.

Bugs -
Even after generating right captcha, sometimes invalid captcha error can come.
